Custom Live Search For Searching Post using Shortcode [live-post-search]

== Description ==

This plugin use for live search for post without page refresh Using Shortcode [live-post-search]


Primary Features
*  Very easy to use just copy this Shortcode [live-post-search]
*  A front-end Custom Search
*  Live Search without page refresh
*  This plugin is free


== Installation ==

Go to Plugins > Add New.
Search for "Live search for post".
Click "Install Now".

OR

Download the zip file.
Upload the zip file via Plugins > Add New > Upload.
Activate the plugin. Look for "Live Search" in the admin menu.

== Frequently Asked Questions ==

= What is the shortcode? =

use this shortcode:

[live-post-search]
